# 📸 Instruções para Atualizar o Logo

## Como Adicionar o Novo Logo

### Opção 1: Logo Local (Recomendado)

1. **Coloque o arquivo do logo na pasta `public/`**
   - Nome do arquivo: `LOGO.png` (ou `.jpg`, `.svg`, etc.)
   - Caminho: `public/LOGO.png`

2. **O código já está configurado para usar `/LOGO.png`**
   - O Vite serve arquivos da pasta `public/` na raiz do site
   - Não é necessário alterar o código

### Opção 2: Logo Online (URL)

Se o logo estiver hospedado online, você pode fornecer a URL e eu atualizo as referências no código.

## Locais onde o logo será atualizado:

- ✅ Navbar (menu superior)
- ✅ Footer (rodapé)
- ✅ Meta tags Open Graph (para compartilhamento em redes sociais)
- ✅ Favicon (ícone do navegador)

## Após adicionar o logo:

1. Execute o build:
   ```bash
   npm run build
   ```

2. Faça upload dos arquivos atualizados na Hostinger

---

**Nota:** Se você já tem o arquivo do logo, coloque-o na pasta `public/` com o nome `LOGO.png` e me avise para que eu possa atualizar todas as referências no código.
