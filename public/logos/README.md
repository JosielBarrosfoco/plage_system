# Logos PLAGE System

Esta pasta contém as imagens do logo da PLAGE System.

## Estrutura de arquivos esperada:

- `logo-light.png` - Logo com fundo claro/transparente (usado em fundos escuros)
- `logo-dark.png` - Logo com fundo escuro (usado em fundos claros)  
- `logo-circle.png` - Logo circular com fundo
- `logo-full.png` - Logo completo com "PLAGE SYSTEM"

## Onde cada variante é usada:

- **logo-light.png**: Header (topo do site), Footer (rodapé)
- **logo-dark.png**: Seções com fundo claro (se necessário)
- **logo-circle.png**: Ícones, favicon alternativo
- **logo-full.png**: Página inicial (hero section), materiais promocionais

## Formatos recomendados:

- PNG com fundo transparente (para light/dark)
- Resolução: mínimo 800px de largura
- DPI: 300 para materiais impressos, 72-150 para web
